群组管理

POST /api/group/add-members Plus版 - 添加群成员
接口说明

向指定群组添加新成员。

请求参数
参数名 类型 必需 说明
group / group_name string 是 群组名称（支持两种参数名）
members array / string 是 要添加的成员列表（支持数组或单个字符串）
reason string 否 邀请理由
响应格式
{
    "success": true,
    "message": "成功添加群成员",
    "data": {
        "added_members": ["用户1", "用户2"],
        "failed_members": []
    }
}
GET /api/group/get-members Plus版 - 获取群成员列表
接口说明

获取指定群组的成员列表。

请求参数
参数名 类型 必需 说明
who string 是 群组名称
响应格式
{
    "success": true,
    "message": "获取群成员列表成功",
    "data": {
        "group_name": "测试群",
        "members": [
            {
                "name": "群主",
                "role": "owner"
            },
            {
                "name": "成员1",
                "role": "member"
            }
        ]
    }
}
GET /api/group/get-recent-groups Plus版 - 获取最近群聊
接口说明

获取最近群聊名称列表。

响应格式
{
    "code": 0,
    "message": "获取最近群聊成功",
    "data": {
        "groups": ["群聊1", "群聊2", "群聊3"]
    }
}
GET /api/group/get-contact-groups Plus版 - 获取通讯录群聊
接口说明

获取通讯录中的群聊列表。

请求参数
参数名 类型 必需 说明
speed number 否 滚动速度，默认1
interval number 否 滚动间隔，默认0.1
响应格式
{
    "code": 0,
    "message": "获取通讯录群聊成功",
    "data": {
        "groups": ["群聊1", "群聊2", "群聊3"]
    }
}
